# ReadMe for Recidivism Analysis Project

## There are three main files/folders within this repository
1. Data - includes the data used for this project
2. DecisionTrees - includes the implementation of a decision tree algorithm
    This file is in a jupyter notebook, run the code blocks in order to see the code output.
3. Logisitic Regression - includes the implementation of a logistic regression algorithm
    This file is in a jupyter notebook, run the code blocks in order to see the code output.
4. random_forest.py - a python file that runs the random forest model with corresponding metric outputs and plots.
